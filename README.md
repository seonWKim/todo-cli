# TODO-CLI 

todo-cli is a simple command line tool to manage your todo list. 

## Features 

- [ ] `tc -a <task>`: Add a task to the list 
- [ ] `tc -l`: List all tasks 
- [ ] `tc -d <task>`: Mark a task as done
- [ ] `tc -r <task>`: Remove a task from the list
- [ ] `tc -h`: Show help
